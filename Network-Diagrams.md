# Network Diagrams

Network documentation standards and infrastructure inventory procedures.

## Network Overview

Network documentation should include:

- Internet connectivity
- Firewalls
- Switches
- Wireless access points
- Servers
- Printers
- End-user devices

## Network Segmentation

### Staff Network

- Employee workstations
- Business applications
- Corporate resources

### Guest Network

- Internet access only
- Isolated from internal resources

### Production Network

- Manufacturing devices
- CNC systems
- Operational technology

### Printer Network

- Network printers
- Print services

## Documentation Requirements

### IP Address Inventory

- Device name
- IP address
- VLAN assignment
- Location
- Owner

### Device Inventory

- Firewalls
- Switches
- Access points
- Servers
- Printers

## Security Considerations

- VLAN separation
- Least privilege access
- Firewall controls
- Secure management access
- Network monitoring

## Technologies

- FortiGate
- VLANs
- TCP/IP
- DNS
- DHCP
- Network Switching
- Wireless Networks
